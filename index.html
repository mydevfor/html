HTML Tags Presentation

This is an introduction to essential HTML tags and their descriptions.

<!DOCTYPE>

The <!DOCTYPE> declaration defines the document type and version of HTML. For example:

<!DOCTYPE html>

<html> and </html>

The <html> tag is the root element of an HTML document. It contains all other elements. The closing tag is </html>.


<html>
    ...
</html>

<head> and </head>

The <head> tag contains metadata, scripts, styles, and title of the document. The closing tag is </head>.


<head>
    <title>Page Title</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
</head>

<title> and </title>

The <title> tag sets the title of the web page, which appears in the browser tab. The closing tag is </title>.


<title>My Web Page</title>

<body> and </body>

The <body> tag contains the contents of an HTML document, such as text, images, links, etc. The closing tag is </body>.


<body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph.</p>
</body>

<h1> to <h6>

Heading tags <h1> to <h6> are used to define headings, with <h1> as the highest level and <h6> as the lowest. 


<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>
<h4>Sub-sub-subheading</h4>
<h5>Sub-sub-sub-subheading</h5>
<h6>Sub-sub-sub-sub-subheading</h6>

<p> and </p>

The <p> tag defines a paragraph of text. The closing tag is </p>.


<p>This is a paragraph.</p>

<a> and </a>

The <a> tag defines a hyperlink, which links to another page or resource. The href attribute specifies the link destination. The closing tag is </a>. 


<a href="http://example.com">Visit Example.com</a>

<img>

The <img> tag embeds an image in the document. It has attributes like src (source URL) and alt (alternative text). This tag is self-closing and does not need a closing tag.


<img src="image.jpg" alt="Description of image">

<ul>, <ol> and <li>

The <ul> tag defines an unordered list, <ol> defines an ordered list, and <li> defines a list item. These tags need closing tags.


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<div> and </div>

The <div> tag is a container that divides the document into sections. It is often used with CSS for styling. The closing tag is </div>.


<div>
    <p>This is a paragraph inside a div.</p>
</div>

<span> and </span>

The <span> tag is an inline container used to mark up a part of a text or a part of a document. It is used with CSS for styling. The closing tag is </span>.


<span style="color: red;">This text is red.</span>

<center> and </center>

The <center> tag is used to center-align text or other elements. Note: This tag is now obsolete and it's recommended to use CSS for centering elements. The closing tag is </center>.


        <center>
        <p>This text is centered.</p>
        </center>

The <i> tag is used to make text italic. The closing tag is </i>.


        <p>This is <i>italic</i> text.</p>

<em> and </em>

The <em> tag is used to emphasize text. The closing tag is </em>.


        <p>This is <em>emphasized</em> text.</p>

<mark> and </mark>

The <mark> tag is used to highlight text. The closing tag is </mark>. 


        <p>This is <mark>marked</mark> text.</p>

<small> and </small>

The <small> tag is used to make text smaller. The closing tag is </small>. 


        <p>This is <small>smaller</small> text.</p>

<del> and </del>

The <del> tag is used to represent deleted text. The text inside this tag is typically displayed with a strikethrough. The closing tag is </del>.


        <p>This is <del>deleted</del> text.</p>

<ins> and </ins>

The <ins> tag is used to represent inserted text. The closing tag is </ins>. 


        <p>This is <ins>inserted</ins> text.</p>

<sub> and </sub>

The <sub> tag is used for subscript text. The closing tag is </sub>. 


        <p>This is <sub>subscript</sub> text.</p>

<sup> and </sup>

The <sup> tag is used for superscript text. The closing tag is </sup>. 


        <p>This is <sup>superscript</sup> text.</p>

<form> and </form>

The <form> tag is used to create an HTML form for user input. The closing tag is </form>. The form element can contain various input elements. The method attribute specifies how to send form data.

Methods:

  • GET: Appends data to the URL (used for retrieving data).
  • POST: Sends data as a separate entity (used for submitting data).

Example:


    <form action="/submit_form" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br><br>
    <input type="submit" value="Submit">
    </form>

<br>

The <br> tag in HTML is used to insert a line break. It is an empty element, meaning it does not have a closing tag. The
tag is useful when you want to start a new line of text within the same paragraph or block-level element.

<script>

The <script> tag in HTML is used to embed or reference executable code, typically JavaScript.It can be placed within the < head > or < body > sections of an HTML document.The code inside the < script > tag can be written directly within the tag, or it can be linked to an external JavaScript file using the src attribute.

Examples: Inline JavaScript: 
<script>
document.write( "This is an inline JavaScript example." ); 
</script>
External JavaScript: First, create an external JavaScript file, script.js:
// script.js 
        function displayMessage() {
             document.write("This is an inline JavaScript example." );
}
Then, link to this file in your HTML document:
<script src="script.js"></script>
Attributes:
src: Specifies the URL of an external script file.
type: Specifies the MIME type of the script (default is text/javascript).
async: Executes the script asynchronously as soon as it is available.
defer: Executes the script when the page has finished parsing.

Examples with Attributes:


1. Using src Attribute:


<script src ="path/to/your/script.js"></script>

2. Using async Attribute:

<script src ="path/to/your/script.js" async></script>

Using defer Attribute:

<script src ="path/to/your/script.js" defer></script>
Notes: Scripts with the async attribute will be executed as soon as the script is downloaded, without blocking the page rendering. Scripts with the defer attribute will be executed in the order they appear in the document, after the page has finished parsing. Use the <script> tag to enhance your web pages with dynamic content and interactivity through JavaScript.

<link> and </link>

The <link> tag in HTML is used to define the relationship between the current document and an external resource. The closing tag is not required as <link> is a self-closing tag. To add a favicon to your HTML document, you use the <link> tag within the <head> section.

Attributes:

  • href: Specifies the path to the favicon file.
  • rel: Specifies the relationship between the current document and the linked resource. For favicons, this is typically "icon" or "shortcut icon".
  • type: Specifies the MIME type of the linked resource. For favicons, common types include "image/x-icon", "image/png", etc.
  • sizes: Defines the sizes of the icons (e.g., "32x32") to provide different resolutions for different devices.

Example:

  
        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
            <link rel="icon" href="path/to/your/favicon.ico" type="image/x-icon">
            <link rel="icon" type="image/png" sizes="32x32" href="path/to/your/favicon-32x32.png">
            <link rel="icon" type="image/png" sizes="16x16" href="path/to/your/favicon-16x16.png">
            <link rel="icon" type="image/svg+xml" href="path/to/your/favicon.svg">
            <link rel="shortcut icon" href="path/to/your/favicon.ico">
        </head>
        <body>
            <h1>Hello, World!</h1>
        </body>
        </html>

<iframe> and </iframe>

The <iframe> tag in HTML is used to embed another HTML document within the current document. It creates an inline frame, which can be used to display a web page within a web page. The closing tag is </iframe>.

Attributes:

  • src: Specifies the URL of the page to embed.
  • width: Specifies the width of the iframe.
  • height: Specifies the height of the iframe.
  • name: Specifies a name for the iframe, which can be used as the target for links.
  • frameborder: Specifies whether or not to display a border around the iframe.
  • scrolling: Specifies whether or not to allow scrolling within the iframe.
  • allow: Specifies a list of features for the iframe to allow.
  • allowfullscreen: Allows the iframe content to be displayed in fullscreen mode.

Example:


        <!DOCTYPE html>
        <html lang="en">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
        </head>
        <body>
        <h1>Embedded Page Example</h1>
        <iframe src="https://www.example.com" width="600" height="400" frameborder="0" allowfullscreen></iframe>
        </body>
        </html>

<input>

The <input> tag in HTML is used to create interactive controls in a web form that accepts user input. It is a self-closing tag, meaning it does not have a closing tag. Various types of inputs can be specified using the type attribute.

Common Input Types:

  • type="text": Defines a single-line text input field.
  • type="password": Defines a password field, characters are masked.
  • type="email": Defines a field for entering an email address.
  • type="number": Defines a field for entering a number.
  • type="date": Defines a field for selecting a date.
  • type="checkbox": Defines a checkbox.
  • type="radio": Defines a radio button.
  • type="submit": Defines a submit button for submitting the form.
  • type="reset": Defines a reset button for resetting the form fields to their initial values.
  • type="button": Defines a clickable button.
  • type="file": Defines a file-select field and a "Browse" button for file uploads.
  • type="hidden": Defines a hidden input field.
  • type="color": Defines a color picker.
  • type="range": Defines a slider control for selecting a value in a range of values.
  • type="search": Defines a search field (like a site search or Google search).
  • type="tel": Defines a field for entering a telephone number.
  • type="url": Defines a field for entering a URL.

Examples:


        <!DOCTYPE html>
        <html lang="en">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Input Types Example</title>
        </head>
        <body>
        <form>
        <label for="text">Text Input:</label>
        <input type="text" id="text" name="text"><br><br>
        
        <label for="password">Password Input:</label>
        <input type="password" id="password" name="password"><br><br>
        
        <label for="email">Email Input:</label>
        <input type="email" id="email" name="email"><br><br>
        
        <label for="number">Number Input:</label>
        <input type="number" id="number" name="number"><br><br>
        
        <label for="date">Date Input:</label>
        <input type="date" id="date" name="date"><br><br>
        
        <label for="color">Color Picker:</label>
        <input type="color" id="color" name="color"><br><br>
        
        <label for="range">Range Slider:</label>
        <input type="range" id="range" name="range" min="1" max="100"><br><br>
        
        <label>Checkbox:</label>
        <input type="checkbox" id="checkbox1" name="checkbox1" value="1"> Option 1
        <input type="checkbox" id="checkbox2" name="checkbox2" value="2"> Option 2<br><br>
        
        <label>Radio Button:</label>
        <input type="radio" id="radio1" name="radio" value="1"> Option 1
        <input type="radio" id="radio2" name="radio" value="2"> Option 2<br><br>
        
        <input type="submit" value="Submit">
        <input type="reset" value="Reset">
        </form>
        </body>
        </html>

<video> and </audio>

The <video> and <audio> tags in HTML are used to embed video and audio content in a web page, respectively. Both elements have attributes that provide control over playback, source files, and compatibility with various devices and browsers.

<video> tag:

The <video> tag is used to embed video content. It requires at least one <source> element to specify the video files. The closing tag is </video>.

Attributes:

  • src: Specifies the URL of the video file. (Optional, can be provided in the <source> element)
  • controls: Displays video controls such as play, pause, and volume.
  • autoplay: Starts playing the video as soon as it is ready.
  • loop: Plays the video repeatedly.
  • muted: Mutes the audio in the video.
  • width and height: Specifies the dimensions of the video player.
  • poster: Specifies an image to be shown while the video is downloading or until the user hits the play button.

Example:


        <video width="640" height="480" controls>
        <source src="movie.mp4" type="video/mp4">
        <source src="movie.ogg" type="video/ogg">
        Your browser does not support the video tag.
        </video>

<audio> tag:

The <audio> tag is used to embed audio content. It requires at least one <source> element to specify the audio files. The closing tag is </audio>.

Attributes:

  • src: Specifies the URL of the audio file. (Optional, can be provided in the <source> element)
  • controls: Displays audio controls such as play, pause, and volume.
  • autoplay: Starts playing the audio as soon as it is ready.
  • loop: Plays the audio repeatedly.
  • muted: Mutes the audio.

Example:


        <audio controls>
        <source src="you're audio name.mp3" type="audio/mpeg">
        Your browser does not support the audio tag.
        </audio>

<header>, <main>, <footer>, and <section>

These HTML5 semantic elements are used to define the structure and organization of a web page. They provide meaning to the content, making it easier for search engines and screen readers to understand the document.

<header> Tag:

The <header> tag is used to represent the introductory content or navigational links of a section or page. It typically contains headings, logos, or other introductory content.

Example:


        <header>
        <h1>Welcome to My Website</h1>
        <nav>
        <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#contact">Contact</a></li>
        </ul>
        </nav>
        </header>

<main> Tag:

The <main> tag is used to encapsulate the dominant content of a document. This content is unique and should not include any content that repeats across documents (like headers, footers, or sidebars).

Example:


        <main>
                <article>
            <h2>Article Title</h2>
            <p>This is the content of the article.</p>
                </article>
        </main>

<footer> Tag:

The <footer> tag is used to define the footer of a document or section. It typically contains information about the author, copyright information, or links to related documents.

Example:


        <footer>
                <p>© 2024 My Website. All rights reserved.</p>
        <nav>
        <ul>
            <li><a href="#privacy">Privacy Policy</a></li>
            <li><a href="#terms">Terms of Service</a></li>
        </ul>
        </nav>
        </footer>

<section> Tag:

The <section> tag is used to define a section in a document. It represents a thematic grouping of content, typically with a heading.

Example:


        <section>
        <h2>Section Title</h2>
        <p>This is the content of the section.</p>
        </section>
        
        <section>
        <h2>Another Section Title</h2>
        <p>This is the content of another section.</p>
        </section>

<meta> Tags

The <meta> tags in HTML provide metadata about the HTML document. Metadata will not be displayed on the page but will be machine-readable. Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata. They are always placed inside the <head> tag.

Common <meta> Attributes:

  • name: Specifies the name of the metadata
  • content: Provides the value or content of the metadata
  • charset: Specifies the character encoding for the HTML document
  • http-equiv: Provides an HTTP header for the information/value of the content attribute

Examples of Common <meta> Tags:

  • <meta charset="UTF-8">: Specifies the character encoding for the document
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Provides instructions for controlling the page's dimensions and scaling
  • <meta name="description" content="A brief description of the page">: Describes the content of the page
  • <meta name="keywords" content="HTML, CSS, JavaScript">: Provides a list of keywords relevant to the content of the page
  • <meta name="author" content="Author Name">: Specifies the author of the document
  • <meta http-equiv="refresh" content="30">: Refreshes the document every 30 seconds
  • <meta http-equiv="X-UA-Compatible" content="IE=edge">: Sets the document mode in Internet Explorer to edge mode

Example:


        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <meta name="description" content="A brief description of the page">
            <meta name="keywords" content="HTML, CSS, JavaScript">
            <meta name="author" content="Author Name">
            <meta http-equiv="refresh" content="30">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <title>Document</title>
        </head>
        <body>
                <h1>Hello, World!</h1>
        </body>
        </html>

Open Graph Meta Tags:

Open Graph meta tags are used to improve the display of shared content on social media platforms. They provide more control over how URLs are displayed when shared.

  • <meta property="og:title" content="The title of the content">: Specifies the title of the content
  • <meta property="og:description" content="A brief description of the content">: Describes the content
  • <meta property="og:image" content="URL to the image">: Specifies an image URL that represents the content
  • <meta property="og:url" content="URL of the content">: Specifies the canonical URL of the content

Example:


        <meta property="og:title" content="The title of the content">
        <meta property="og:description" content="A brief description of the content">
        <meta property="og:image" content="URL to the image">
        <meta property="og:url" content="URL of the content">

Twitter Card Meta Tags:

Twitter Card meta tags are used to control how content is displayed in Twitter tweets.

  • <meta name="twitter:card" content="summary">: Specifies the type of card to be displayed (e.g., summary, photo, video)
  • <meta name="twitter:title" content="The title of the content">: Specifies the title of the content
  • <meta name="twitter:description" content="A brief description of the content">: Describes the content
  • <meta name="twitter:image" content="URL to the image">: Specifies an image URL that represents the content
  • <meta name="twitter:site" content="@username">: Specifies the Twitter username of the content creator

Example:


        <meta name="twitter:card" content="summary">
        <meta name="twitter:title" content="The title of the content">
        <meta name="twitter:description" content="A brief description of the content">
        <meta name="twitter:image" content="URL to the image">
        <meta name="twitter:site" content="@username">

Additional Meta Tags:

  • <meta name="theme-color" content="#ffffff">: Specifies the theme color for the browser's address bar (supported on mobile browsers)
  • <meta name="robots" content="index, follow">: Specifies the behavior of search engine crawlers (e.g., index or noindex, follow or nofollow)

Example:


        <meta name="theme-color" content="#ffffff">
        <meta name="robots" content="index, follow">
